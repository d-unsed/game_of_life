# Conway's game of Life

### Features

 - No mutability
 - No conditional statements
 - No primitive types except for constructors
 - No state inside `Cell`s. Use `DeadCell` and `AliveCell` structures with separated behavior.
 - Infinite grid

### Usage

1. Clone repository
2. Run executable
```bash
cargo run --release
```
3. Enjoy :space_invader:
